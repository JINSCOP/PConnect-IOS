# PConnect-IOS

#### 项目介绍
这是PConnect项目的IOS客户端项目

#### 软件架构
* 使用Cocoapods管理依赖
* 软件的主体结构使用Swift编码，二级或者三级栏目/子页面等可能采用Webkit+H5,使用javascriptCore进行网页和本地交互
* 采用Sqllite做本地数据缓存
* 采用的第三方组件[链接](./Podfile)
* 知识库文本类全部采用Markdown作为主要书写规范
